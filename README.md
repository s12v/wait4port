# wait4port

Wait for an open port with 60 seconds timeout

## Usage

```
bash <(curl -s https://raw.githubusercontent.com/s12v/wait4port/master/wait4port.sh) host:port
```

## Return codes

* `0` - success
* `1` - timeout
